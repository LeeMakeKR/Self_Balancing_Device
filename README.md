# Self_Balancing_Device

## 프로젝트 개요
자이로 센서와 밸런스 휠 모터를 이용하여 모서리로 서는 자가 균형 장치를 개발하는 프로젝트입니다.

## 개발 계획
이 프로젝트는 단계별로 업그레이드하여 더욱 정밀한 균형 제어를 목표로 합니다:

### 1단계: 1축 균형 장치
- **목표**: 한 축 방향의 균형 제어
- **구성 요소**:
  - 자이로 센서 (각속도 감지)
  - 밸런스 휠 모터 1개
  - 마이크로컨트롤러 (Arduino 기반)
- **기능**: 기본적인 균형 유지 및 제어 알고리즘 구현

### 2단계: 2축 균형 장치  
- **목표**: 두 축 방향의 균형 제어
- **구성 요소**:
  - 개선된 자이로 센서 (IMU)
  - 밸런스 휠 모터 2개 (X, Y축)
  - 향상된 제어 보드
- **기능**: 2차원 평면에서의 정밀한 균형 제어

### 3단계: 3축 균형 장치
- **목표**: 완전한 3차원 균형 제어
- **구성 요소**:
  - 고정밀 9축 IMU 센서
  - 밸런스 휠 모터 3개 (X, Y, Z축)
  - 고성능 제어 시스템
- **기능**: 모든 방향에서의 완벽한 자세 제어 및 안정화

## 기술적 특징
- **센서**: 자이로스코프를 통한 실시간 각속도 측정
- **액추에이터**: 밸런스 휠의 관성 모멘트를 이용한 반작용 제어
- **제어**: PID 제어 알고리즘 기반 균형 유지
- **목표**: 모서리 위에서 안정적으로 서는 능력 구현
